## شرح القيم:
- الجنس: 1 = ذكر، 0 = أنثى
- نوع ألم الصدر: 0 = غير معروف، 1 = ذبحة صدرية نموذجية، 2 = ذبحة صدرية غير نموذجية، 3 = ألم غير ذبحي، 4 = بدون أعراض
- تخطيط القلب: 0 = طبيعي، 1 = شذوذ ST-T، 2 = تضخم البطين الأيسر
- ذبحة جهدية: 0 = لا، 1 = نعم
- ميل ST: 0 = صاعد، 1 = مستوي، 2 = هابط
- الثاليوم: 0 = غير معروف، 1 = طبيعي، 2 = عيب ثابت، 3 = عيب قابل للإصلاح

يمكنك نسخ القيم وتجربتها في الحاسبة للتأكد من ظهور كل مستوى خطر بشكل صحيح.
